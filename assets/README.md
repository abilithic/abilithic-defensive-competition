# assets/

Taruh file branding & screenshot di sini — README.md di root sudah mereferensikan
nama file ini (sama seperti pola di `abilithic-recon` & `abilithic-scan`), jadi begitu
file ini ada, gambar di README otomatis muncul tanpa perlu edit lagi.

| File | Dipakai di | Ukuran disarankan |
|---|---|---|
| `abilithic-icon-256.png` | Logo di header README | 256×256 |
| `abilithic-horizontal-1200.png` | Banner di bawah tagline | ~1200×400 |
| `screenshot-leaderboard.png` | Papan skor publik (`/`) | 1280×800 (16:10) |
| `screenshot-admin.png` | Konsol panitia (`/admin`) | 1280×800 |
| `screenshot-kiosk.png` | Companion app peserta (kiosk) | 460×880 (potret) |

Format PNG, ukuran file idealnya < 500 KB per gambar (kompres dulu bila perlu)
agar README tetap cepat dimuat di GitHub.
