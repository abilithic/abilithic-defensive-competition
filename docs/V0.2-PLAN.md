# abilithic DHC — Rencana v0.2 (Profesionalisasi)

> v0.1 sudah jalan end-to-end. v0.2 fokus: **rasa profesional** untuk peserta & panitia.

## Sasaran

### A. Agent → Aplikasi Kiosk (peserta zero-setup)
Peserta **tidak** buka terminal/browser/localhost. Pengalaman baru:
- VM boot → **auto-login** → langsung muncul **aplikasi fullscreen ber-brand** (kiosk).
- Layar 1: form registrasi (Nama, Sekolah, Kode Sesi).
- Setelah START: berubah jadi **live dashboard** (skor, timer, checklist, hint).
- Semua sudah terpasang & auto-start di dalam **image OVA** → peserta cukup boot & isi nama.
- Teknis: bungkus UI web-lokal jadi **jendela native** (pywebview) + **autostart fullscreen**;
  desktop Ubuntu disembunyikan (mode kiosk). Dikemas ke OVA final.

### B. Web → Console Profesional
1. **Login persisten & aman** — ganti password-di-memori jadi **sesi cookie httpOnly** (tak hilang saat refresh, password tak dikirim tiap request). *(Opsi lanjut: Supabase Auth multi-admin.)*
2. **Kelola event & peserta** — hapus/arsip sesi, daftar peserta per sesi, diskualifikasi/hapus peserta, dialog konfirmasi.
3. **Redesain tampilan** — sistem desain konsisten, branding, badge status, toast, loading & empty state, mode layar-besar untuk proyektor.
4. **Evidence Viewer + Export** — lihat snapshot before/after per peserta, export hasil CSV.
